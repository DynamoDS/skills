Add marketplace
/plugin marketplace add DynamoDS/skills
Install plugin
/plugin install dynamo-skills

Skills documentation ↗

Copy skills into your repo

Drop skill folders into .github/skills/, .agents/skills/, or .claude/skills/ for project access, or ~/.copilot/skills/ for personal skills across all projects. Copilot discovers them automatically.

In the CLI, use /skill-name to invoke a skill explicitly, or /skills list to see what's available.

Agent skills documentation ↗

Copy skills into your repo

Drop skill folders directly into .github/skills/, .agents/skills/, or .claude/skills/ — VS Code discovers those automatically, no settings required.

To keep skills in sync as a git submodule instead, add the repo and point chat.skillsLocations at the skills/ subfolder in your .vscode/settings.json. Open in VS Code

Shared skills documentation ↗

Copy skills into your project

Drop skill folders into .cursor/skills/ in your repo, or ~/.cursor/skills/ for global (user-wide) access. Cursor discovers them automatically.
